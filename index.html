Kelompok 8 PHP

Membangun solusi web yang inovatif dan profesional dengan teknologi PHP terdepan untuk menciptakan pengalaman digital yang luar biasa.

TENTANG KAMI

Kami adalah tim yang berdedikasi dalam mengembangkan aplikasi web menggunakan PHP. Dengan kombinasi kreativitas, teknologi terdepan, dan semangat kolaborasi, kami berkomitmen untuk menciptakan solusi digital yang tidak hanya fungsional, tetapi juga memberikan pengalaman pengguna yang luar biasa.

Setiap anggota tim membawa keahlian unik yang saling melengkapi, mulai dari pengembangan backend yang robust hingga desain antarmuka yang menarik dan user-friendly.

DOKUMENTASI PEMBELAJARAN

Pertemuan 1 - Pengenalan PHP Dasar

15 September 2024
Dr. Ahmad Sutisna

Pemateri

Sarah Williams

Moderator

Rangkuman Materi
  • Pengenalan PHP: Sejarah, karakteristik, dan keunggulan PHP sebagai bahasa pemrograman server-side
  • Sintaks Dasar: Tag pembuka
  • Tipe Data: String, Integer, Float, Boolean, Array, dan Object
  • Operator: Aritmatika, perbandingan, logika, dan assignment
  • Struktur Kontrol: If-else, switch-case, dan penggunaan kondisi
  • Environment Setup: Instalasi XAMPP, konfigurasi server local, dan testing pertama
Tugas Pertemuan 1
1. Instalasi dan Setup Environment

Install XAMPP dan buat halaman PHP pertama yang menampilkan "Hello World" dengan informasi sistem menggunakan phpinfo().

2. Latihan Variabel dan Tipe Data

Buat script PHP yang mendeklarasikan berbagai tipe data (string, integer, float, boolean, array) dan tampilkan menggunakan var_dump().

3. Aplikasi Kalkulator Sederhana

Buat kalkulator sederhana yang dapat melakukan operasi dasar (tambah, kurang, kali, bagi) dengan menggunakan operator PHP.

Deadline: 22 September 2024, 23:59 WIB

Pertemuan 2 - Struktur Kontrol dan Fungsi

22 September 2024
Prof. Indira Sari, M.Kom

Pemateri

Michael Chen

Moderator

Rangkuman Materi
  • Perulangan (Loop): For, while, do-while, dan foreach untuk array
  • Nested Loop: Perulangan bersarang dan aplikasinya dalam membuat pola
  • Fungsi Built-in: String functions, array functions, dan mathematical functions
  • User-defined Functions: Cara membuat fungsi sendiri, parameter, dan return value
  • Scope Variable: Global, local, dan static variables
  • Include dan Require: Cara mengorganisir code dengan file terpisah
Tugas Pertemuan 2
1. Pattern Generator

Buat program yang menghasilkan berbagai pola (segitiga, persegi, diamond) menggunakan nested loop dengan input dari user.

2. Library Management System

Buat sistem sederhana untuk mengelola buku dengan fungsi: tambah buku, cari buku, tampilkan semua buku menggunakan array dan functions.

3. String Manipulation Tool

Buat tool yang dapat melakukan berbagai operasi string: reverse, count words, find and replace, uppercase/lowercase conversion.

4. Modular Code Structure

Reorganisasi tugas sebelumnya dengan memisahkan fungsi-fungsi ke file terpisah menggunakan include/require.

Deadline: 29 September 2024, 23:59 WIB

Pertemuan 3 - Array dan Manipulasi Data

29 September 2024
Dr. Rizki Pratama

Pemateri

Emily Davis

Moderator

Rangkuman Materi
  • Array Multidimensi: Konsep dan implementasi array 2D, 3D, dan nested array
  • Array Functions: sort(), rsort(), array_push(), array_pop(), array_merge(), array_slice()
  • Associative Array: Key-value pairs dan penggunaannya dalam data struktur
  • Array Iteration: foreach, array_map(), array_filter(), array_reduce()
  • JSON Handling: json_encode() dan json_decode() untuk pertukaran data
  • Data Validation: Teknik validasi data input dan sanitasi
Tugas Pertemuan 3
1. Student Grade Management

Buat sistem manajemen nilai siswa menggunakan multidimensional array dengan fitur: input nilai, hitung rata-rata, ranking, dan export ke JSON.

2. Product Inventory System

Buat sistem inventori produk dengan associative array yang memiliki fitur: CRUD produk, search, filter berdasarkan kategori/harga.

3. Data Processing Tool

Buat tool yang dapat memproses data dari file JSON, melakukan filtering, sorting, dan menghasilkan laporan statistik.

Deadline: 6 Oktober 2024, 23:59 WIB

Pertemuan 4 - Form Handling dan Validasi

6 Oktober 2024
Ms. Diana Kusuma

Pemateri

Alex Johnson

Moderator

Rangkuman Materi
  • HTTP Methods: Perbedaan GET dan POST, kapan menggunakan masing-masing
  • Form Processing: $_GET, $_POST, $_REQUEST untuk mengambil data form
  • Input Validation: Server-side validation, filter_var(), dan custom validation functions
  • Security Practices: XSS prevention, SQL injection basics, data sanitization
  • File Upload: $_FILES, validasi file, move_uploaded_file()
  • Session Management: session_start(), $_SESSION, login/logout system
Tugas Pertemuan 4
1. Registration Form System

Buat sistem registrasi lengkap dengan validasi email, password strength, dan konfirmasi password. Include error handling dan success messages.

2. Contact Form with File Upload

Buat contact form yang memungkinkan user upload file (image/document) dengan validasi tipe file, ukuran, dan sanitasi input.

3. Simple Login System

Implementasikan sistem login sederhana menggunakan session dengan fitur remember me dan logout functionality.

Deadline: 13 Oktober 2024, 23:59 WIB
