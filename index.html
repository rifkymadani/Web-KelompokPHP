TENTANG KAMI

PHP adalah bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. PHP pertama kali dikembangkan oleh Rasmus Lerdorf pada tahun 1994. PHP bisa dijalankan di server web (Linux/Windows/macOS). Sangat umum di hosting web, CMS (WordPress, Drupal), aplikasi web back-end. Bahasa PHP banyak digunakan karena Mudah dipelajari, cepat untuk pengembangan web, banyak hosting & library, ekosistem besar & aplikasi produksi. Kode PHP ditulis dalam tag ', dieksekusi di server, lalu mengirim HTML ke browser.
